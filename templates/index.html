Quick Start
1. Get API Key

Contact your administrator to obtain an API key for authentication.

2. Upload PDF

Use the /api/upload endpoint to upload your PDF files.

3. Process

Call any processing endpoint with your uploaded files to perform operations.

Example Request
curl -X POST \
  -H "X-API-Key: your-api-key" \
  -H "Content-Type: application/json" \
  -d '{"files": ["document1.pdf", "document2.pdf"]}' \
  /api/merge
